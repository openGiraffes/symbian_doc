Store

Store provides structured data storage.

    Provides structured data storage, using networks of streams.
  • File Stores
    Provides permanent, direct and dictionary file stores as well as file stream stores.
  • Stores Guide
    Provides persistent, embedded, dictionary, and memory stores.
  • Store Streams
    Provides swizzles, store streams, stream dictionaries, and store maps.
  • Streaming
    Provides various streams and stream operations such as read, write, internalize, and externalize.
