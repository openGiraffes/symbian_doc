Imaging Frameworks Component

The Imaging Framework component provides facilities to an image, to perform the image conversion function, the image processing function, the image transformation function, the image display function, the media client function, the JPEG Exif plug-in function, the Exif Utility function, the Gifscaler function, the bitmap transform function and the JPEG image frame plug-in function.
