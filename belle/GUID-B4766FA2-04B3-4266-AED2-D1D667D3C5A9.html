Symbian C++ API reference

The Symbian C++ API reference provides documentation for the Symbian platform classes and header files.

The Symbian C++ reference is organized according to the Symbian Platform System Model.

Note: There are a number of nodes in the table of contents that do not display a topic. The recommended way to find classes and methods in the API reference is to use the “Search” option.
