Standard C++ Library (STLport v5)

A Standard C++ environment with IOStream and STL (Standard Template Library) libraries enables you to develop or port Standard C++ applications or libraries quickly and easily onto the Symbian platform. The topics in this guide provide a brief overview of Standard C++ support on the Symbian platform and also describe how to develop applications or libraries based on Standard C++.

  • Migrating from STLport v4 to v5
    If you are using STLport v4, it is recommended that you migrate to the latest version of Standard C++ Library (STLport v5 ) available from Symbian^3 to make use of it's latest features.
