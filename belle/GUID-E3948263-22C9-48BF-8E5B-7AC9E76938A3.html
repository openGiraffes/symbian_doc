Tabs

Tabs are used to categorize application content into different views. For design purposes, it is recommended to use icons in Tabs. It is also recommended that you use a maximum of four tabs. If more are needed, the next tab (which cannot be fit into the view) will be partially shown to indicate to the user there are additional tabs. The user can touch the tab and the corresponding view will open.
