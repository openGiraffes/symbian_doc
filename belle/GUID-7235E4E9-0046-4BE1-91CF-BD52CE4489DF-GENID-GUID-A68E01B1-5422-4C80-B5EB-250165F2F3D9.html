Message query

When a query is too long to fit on one screen, a Message query can be used. A Message query can contain a link text that can be highlighted and opened.

By default, the left softkey is OK, but with a link in the query the left softkey opens the target view of the highlighted link text. The Selection key also opens the link target view when the link is highlighted. There should be no more than three links in a single query.
