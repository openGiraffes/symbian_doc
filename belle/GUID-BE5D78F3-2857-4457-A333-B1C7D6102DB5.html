Category: Layout panic

UI resources layout panics.

0

The panic occurs when the window component layout has a negative value.

1

The panic occurs when the window component layout has a offset value that is out of range.

2

The panic occurs when the Aknlayout scalable decode support has an illegal line index

3

The panic occurs when the Aknlayout has a multiline offset out of range.
