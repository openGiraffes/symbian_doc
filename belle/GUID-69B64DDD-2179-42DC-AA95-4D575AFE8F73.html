Category: Font provider

Panic codes for use in Font Provider.

0

The panic occurs when the utility routine to apply the results of a matched font provider line at index aFontIndex into a TFontSpec object is out of range.

1

The panic occurs when the FontProvider component has some missing layout information.

2

The panic occurs when the data structure is out of range.

3

The panic occurs when the MatchIsExact fails to return the font specified.

4

The panic occurs when the specified font is too big.
