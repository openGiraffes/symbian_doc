Editing indicators

The status of the editor is displayed using graphic indicators. These provide the user with information about the following:

  • Editing mode (numeric/alphanumeric, language-specific modes)

  • Character case

  • Predictive Text status

  • Available space

Indicators are usually shown in the Touch Input window, but when the editor is in the Main pane, the indicators are located in the Status bar. If the Status bar is not available, editing indicators are visible in split-input.

A Data query that resides in a pop-up window has its indicators within the window, but when the input is open the indicators are shown there.
