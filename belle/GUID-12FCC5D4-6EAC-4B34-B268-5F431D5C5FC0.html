Memory

One or several memory cards can be added to a Symbian Belle product to increase the available memory. Device may also have internal non-removable memory or internal mass storage — that is, multiple drives are supported.

Users can select the memory that will be used when, for example, performing saving or moving operations. Usually this memory-selection list query is shown immediately after the user has selected an operation to perform.

Some applications may need a setting that defines the used memory in advance.
