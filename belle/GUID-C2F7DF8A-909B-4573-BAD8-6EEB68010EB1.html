Category: AVKON-EIKCOCTL

The panic category for AVKON-EIKCOCTL extension.

0

The panic occurs when the column list layout has an error.

1

The panic occurs when there is a invalid use of list box shortcuts.

2

The panic occurs when the line break has unexpected spaces. This occurs in debug mode only.

3

The panic occurs when the scroll bar extensions are not created.

4

The panic occurs when the list box item draw box is not created.

5

The panic occurs when the object is not fully constructed.
