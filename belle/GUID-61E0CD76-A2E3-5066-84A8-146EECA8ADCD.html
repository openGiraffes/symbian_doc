IP Socket Server

Describes IP Socket Server which provides a generic interface to communications protocols through communication end points known as sockets. The Sockets Server is also known as ESock. Esock is the name of the library that implements the Sockets Server.

You can improve the communication between your PC and the Symbian device by configuring comms components to create a low dependency IP stack. A low dependency IP stack simplifies communication between the test tools running on your PC and the Symbian device. This involves configuring ESock, NT RAS, CommDB and System Starter.
