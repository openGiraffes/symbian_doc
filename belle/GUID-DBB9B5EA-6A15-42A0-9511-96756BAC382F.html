Rights object acquisition protocol

The rights object acquisition protocol (ROAP) defined in OMA DRM v2.0 is the common name for a suite of DRM security protocols between a rights issuer and a DRM agent in a device.

The protocol suite contains the following protocols:

  • A 4-pass protocol for registration of a device with a rights issuer and two protocols for the device to request and acquire rights objects.

  • A 2-pass rights object acquisition protocol for the request and delivery of a rights object.

  • A 1-pass rights object acquisition protocol for the delivery of a rights object from a rights issuer to a device. For example, messaging and push.
