Application Utilities

The Application Utilities is a collection of libraries that are widely used by the high level subsystems and frameworks of the Symbian platform. It provides libraries such as Basic Application Framework Utilities (BAFL) and Base Services Utility Library (BSUL).

  • Application Utilities Overview
    Application utilities is a collection of libraries that provides low level system and application utility APIs to the platform.
