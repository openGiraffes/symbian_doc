Display

The Symbian Belle user interface display specifications are as follows:

  • The common UI components have ready-made scalability, meaning they can be adjusted to different display resolutions and orientations.

    Supported resolutions:

    • • 640 x 360 (nHD) portrait and landscape orientations

  • Square pixels.

  • Color capability (16 million colors preferred)

    Note: Graphics should use 16-bit color depth.
