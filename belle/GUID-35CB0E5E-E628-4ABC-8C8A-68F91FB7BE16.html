Category: CoreHCIBase

Panic codes for Core HCI Base.

1

The panic occurs if the bitmap of channels to be set as bad are busy.

2

The panic occurs if the length of the command frame is greater than the number of valid octets.

3

The panic occurs if the command is not finalized.

4

The panic occurs if the data of the frame length is invalid.

5

The panic occurs if the command data of the frame length is invalid.

6

The panic occurs if the ACL data of the frame length is invalid.

7

The panic occurs if the SCO data of the frame length is invalid.

8

The panic occurs if the length of the required string is greater than required.
