API specifications

This section provides information on using the touch UI component and tactile feedback APIs.

  • Choice list API
    The Choice list API is used for providing a vertical list of selectable items.
  • Generic button API
    The Generic button API is used for enabling touch support for buttons.
  • Stylus pop-up menu API
    The Stylus pop-up menu API is used for displaying commands in a pop-up menu.
  • Tactile feedback client API
    The Tactile feedback client API is used for producing tactile feedback for different touch events.
  • Title pane touch observer API
    The Title pane touch observer API is used for observing when the title pane has been tapped.
  • Toolbar API
    The Toolbar API is used for enabling quick access to functions by displaying them in a toolbar.
  • Touch UI utilities API
    The Touch UI utilities API is used for detecting long-tap events and showing an animation for them.
