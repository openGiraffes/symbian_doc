Symbian Platform API Specifications welcome page

Welcome to Symbian Platform API Specification Documentation set. This page guides you in navigating the documentation.

NOTE: This content is best viewed using Mozilla Firefox 3 or newer Web browser. Also other browsers work but some of the images may not be shown properly.

API Specification documentation structure

The individual documents in this HTML documentation set consist of mostly authored content. Only those APIs with some specifically authored API specification content are listed, e.g. only selected Symbian Platfrom APIs are covered. Furthermore, QT related APIs are not provided in this docset.

The provided documents are organized hierarchically in the main API Specification index, by corresponding SW Layers and Packages. It is the starting point for navigating the API Specifications in this docset.

Examining large diagrams

Some documents may contain very large diagrams. Even though such diagrams may appear unreadable at first, there is a quick way to open them in full screen to see all the details:

  • Mozilla Firefox: Right-click on top of the diagram to open up the context menu. Select "View Image". The image is opened in full screen mode. Left-click on top of the image to zoom in for detailed view.

  • Internet Explorer: Right-click on top of the diagram to open up the context menu. Select "Open link" or "Open link in new window". The image opens up in full screen mode. Move the cursor to the bottom right corner of the image. A pop-up icon appears. Click it to zoom in for detailed view.
