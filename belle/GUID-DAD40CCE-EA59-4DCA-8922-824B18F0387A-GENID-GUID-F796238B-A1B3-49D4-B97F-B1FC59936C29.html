Multi-selection list query

A multi-selection list query is used when the user needs to be able to select several items from a list at the same time (using Selection key or a tap to mark/unmark the checkbox). See Multi-selection list for a description of a multi-selection list.

The left softkey (OK) is used for accepting the query, and the right softkey is Cancel.
