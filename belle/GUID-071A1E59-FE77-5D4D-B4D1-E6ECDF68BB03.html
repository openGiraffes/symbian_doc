Bluetooth Core Collection

The Bluetooth Core collection contains the SDP and the GAVDP components.

  • Bluetooth Core Collection Overview
    The Bluetooth Core collection contains the SDP and GAVDP components.
  • Bluetooth CSY
    Bluetooth CSY is provided primarily to support legacy serial communications (RS232) emulation over Bluetooth.
  • Bluetooth SDP
    The Bluetooth Service Discovery Protocol (SDP) component enables you to register services to the SDP database and query the database for available services.
