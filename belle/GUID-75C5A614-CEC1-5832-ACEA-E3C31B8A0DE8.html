FreeType Font Rasterizer Component Overview

The FreeType Font Rasterizer is a plug-in for the Font Store. The Font Store automatically loads it at system startup and uses it to rasterize compatible open fonts to native Symbian font bitmaps. This is an optional replaceable component, which means that it may be used or replaced by device creators.

The CBR package name for the FreeType Font Rasterizer component is graphics_freetype.
