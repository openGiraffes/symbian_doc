Category: AknIcon

Panic reasons for the client thread while implementation of class CAknIconSrvSession to AknIconServer.

0

The panic occurs when the pointer to the list of CAknBitmaps is removed.

1

The panic occurs when the client cannot get connected to the server.

2

The panic occurs when CAknIconServer cannot create or retrieve a shared icon.

3

The panic occurs when the icon server cannot find the icon.

4

The panic occurs when the bitmap icon is not loaded.

5

The panic occurs when the parameters specified are not valid. This occurs in debug mode only.

6

The panic occurs when the icon location information is incorrect.

7

The panic occurs when the file provider is corrupt.
