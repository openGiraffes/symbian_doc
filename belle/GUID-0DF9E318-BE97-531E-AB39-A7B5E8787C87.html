User Library and File Server

Provides a framework for accessing file systems.

The User Library and File Server collection of components provides access to user-side code to the services provided by Kernel Architecture 2, and completes low-level operating system support.

  • User Library
    The User Library allows the applications and services that run on a phone to access the functionality provided by the Symbian platform Kernel.
  • File Server
    Symbian platform manages the basic permanent storage of data through file systems. Drives and file systems are common resources used by many programs, and access to them is managed by the File Server. The File Server is flexible, and can use many different file system formats and physical storage media. Applications and services access files, directories, and drives through the File Server client side API.
