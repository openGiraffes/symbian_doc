00018 <p><i>declared in "ogg/ogg.h";</i></p>

00020 <p>Returns the number of packets that are completed on this page.  If the
00021 leading packet is begun on a previous page, but ends on this page, it's 
00022 counted.

00043 <h3>Return Values</h3>
00044 <blockquote>
00045 If a page consists of a packet begun on a previous page, and a new packet
00046 begun (but not completed) on this page, the return will be:<br>
00047 <br>
00048 ogg_page_packets(page) will return 1,<br>
00049 ogg_page_continued(paged) will return non-zero.<br>
00050 <br><br>
00051 If a page happens to be a single packet that was begun on a previous page, and
00052 spans to the next page (in the case of a three or more page packet), the
00053 return will be:<br>
00054 <br>
00055 ogg_page_packets(page) will return 0,<br>
00056 ogg_page_continued(page) will return non-zero.<br>
