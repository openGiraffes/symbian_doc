Data Communications Server

The Data Communications Server collection contains the components that provide the frameworks for sockets communications, and for starting network connections.

  • IP Socket Server
    Describes IP Socket Server which provides a generic interface to communications protocols through communication end points known as sockets. The Sockets Server is also known as ESock. Esock is the name of the library that implements the Sockets Server.
