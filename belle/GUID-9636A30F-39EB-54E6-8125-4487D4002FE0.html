Memory Management

Memory is allocated to programs in chunks, the documentation in this set explains how to allocate, manage and release that memory.

  • Memory Management Overview
    Provides functionality concerned with how memory is allocated for and within programs.
  • Cleanup Support Overview
    Provides idioms to handle program exceptions, and to clean up memory when exceptions occur
  • Memory Allocation Overview
    Provides low-level functionality by which a Symbian platform process can access and manipulate memory areas.
  • Memory Management Concepts
    Provides an introduction to memory management, especially chunks and heaps.
  • Using Memory Allocation
    These documents cover heap management, specifically sharing, walking and switching heaps.
  • More about Cleanup Support
    Provides an introduction to code construction and error management options.
  • Heap Management Guide
    This document describes Heap management.
