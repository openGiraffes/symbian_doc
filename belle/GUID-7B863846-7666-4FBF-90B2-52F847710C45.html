Adding and removing menu items

Add items to the menu by either:

  • Calling CAknStylusPopUpMenu::AddMenuItemL() for each menu item to be added or

  • Creating the entire menu instance from a resource by calling CAknStylusPopUpMenu::ConstructFromResourceL(). This requires a STYLUS_POPUP_MENU resource structure that contains the items in the structure STYLUS_POPUP_MENU_ITEM.

Note: If the menu already contains the maximum number of items, CAknStylusPopUpMenu::AddMenuItemL() will do nothing. The same applies to CAknStylusPopUpMenu::ConstructFromResourceL(): it will only add the maximum number of menu items even if there were more items defined in the resource.

To remove items, use the method CAknStylusPopUpMenu::RemoveItem().
