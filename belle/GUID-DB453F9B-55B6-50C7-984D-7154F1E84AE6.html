ICU Layout Engine Component Overview

The ICU (International Components for Unicode) Layout Engine component is a text shaper plug-in for the Font Store. It performs contextual shaping for specific scripts that require this feature.

The CBR package name for the ICU Layout Engine component is graphics_iculayoutengine.
