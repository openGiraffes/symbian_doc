VibraPool: Game example code

The example demonstrates simple vibration and animation of a ball. It specifies vibra settings ans starts it. This is achieved with the help of CHWRMVibra. Animation is achievied with the help of timer CTimer.

Commands are given in the menu option to start and stop the game.
