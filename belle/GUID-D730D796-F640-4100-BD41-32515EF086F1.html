Browsing

The Symbian platform provides the following browser APIs:

  • Browser Plug-in API: This API allows you to develop browser plug-ins for the Symbian platform.

  • Browser Control API: This API allows you to embed the Symbian platform browser within an application.

The following Symbian platform APIs are also related to browsing:

  • Web Services Framework API

  • Favourites Engine API

Additional information

For general information on browsing options and access to browsing documentation and tools, see:

  • Web Technologies on Nokia Developer

  • Web technologies at the Symbian Foundation.
