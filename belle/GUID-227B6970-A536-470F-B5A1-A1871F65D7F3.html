Bluetooth Guide

Find everything you need to know about the Bluetooth package in this section.

  • Bluetooth Core Collection
    The Bluetooth Core collection contains the SDP and the GAVDP components.
  • Bluetooth Management Collection
    The Bluetooth Management collection contains the Bluetooth Manager and Bluetooth Client components.
  • IrDA PRT Plug-in
    IrDA PRT is a ESock Server protocol plugin that provides infra-red support through a sockets interface.
  • RockPaperScissorsGameSourceCode_S60: Game example code
    The project describes how to create a multiplayer game that uses Bluetooth wireless technology to allow two players to play together.
