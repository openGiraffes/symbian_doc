What is IP Security Policy (IPSecPol)?

This topic describes the IPSecPol.

The IPSecPol Manager Server is a part of the IPSec. It is a process that is independent of the Root Server process. The name of the process is ipsecpol.exe and it has a UID of 01000000. It has a configuration which is loaded from the following location:

private\01000000\IPSecPol.pol

This configuration file permits the addition of Autoload policies which allows the current IPSec policy manager functionality to be extended.
