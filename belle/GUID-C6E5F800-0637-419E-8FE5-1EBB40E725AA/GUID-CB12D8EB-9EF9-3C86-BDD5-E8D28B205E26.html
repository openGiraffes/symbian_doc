Detailed Description

Passed into SymbianStreamAddExtendedObserver, SymbianStreamDeactivateObserver and SymbianStreamRemoveObserver for ESOWF_EventContentReady events. Identifies a unique device/context pair. The values are treated as opaque handles.

See also: SymbianStreamAddExtendedObserver SymbianStreamDeactivateObserver SymbianStreamRemoveObserver
