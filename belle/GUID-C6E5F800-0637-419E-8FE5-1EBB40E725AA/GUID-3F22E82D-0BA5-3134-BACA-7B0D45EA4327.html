UTC Offset update notification constants. When the UTC offset is automatically adjusted for Daylight Saving then it is possible to receive notification using the Publish and Subscribe API with the values defined here. Note that this property is updated by the Time Zone Server as this is the only component that can modify the UTC Offset.

See also: RProperty in e32property.h
