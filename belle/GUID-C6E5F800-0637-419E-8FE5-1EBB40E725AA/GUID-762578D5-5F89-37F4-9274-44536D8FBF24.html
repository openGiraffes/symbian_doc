NONSHARABLE_CLASS(CLbsSuplPushRec)

The CLbsSuplPushRec class provides an interface to allow SUPL INIT messages to be received from SMS or WAP Push frameworks. It is used by the SUPL Protocol Module. SUPL INIT messages are sent into the LBS subsystem using the CLbsSuplPush class.
