KTelMatchDigits

const TUint32KTelMatchDigits

Allows a customer to variate the number of digits to be used in contact matching. If Dynamic Matching is used, more digits are used for matching if they are available in both numbers being compared (2 more digits is the limit). If Dynamic Matching is not used then count of digits is used on to match number. Legal values: 7-11. Default value: 7.

KTelDynamicMatching

const TUint32KTelDynamicMatching

Allows a customer to variate if Dynamic Matching is used in contacts matching. Legal values: 1(On) and 0(Off). Default value: 1(On).
