Cellular Baseband Services Guide

Cellular Baseband Services provide a common interface to telephony services for applications and higher level platform components. The framework is independent of the underlying network services.

  • SMS Protocols
    This section describes the components related to the SMS protocols.
  • Telephony Utilities
    The Telephony Utilities is a collection of components that provides small utility libraries related to telephony. The collection also includes the component that implements the telephony API for applications.
