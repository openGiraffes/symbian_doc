Ogg Vorbis codec Overview : Basic audio codecs

The open-source Ogg Vorbis codec is available for pre symbian^2 , symbian ^2 and later . The available codec can be easily built as well as used in the symbian environment. These codec can play ogg vorbis content. This set is an add on to the basic audio codecs .
