Zsh Command Syntax

Zsh supports the following commands of which those supported by Symbian are listed below:

Note: Out of all the commands listed here only find, grep, ps, unzip and zip are external commands and others are built-in commands.

The information related to the above commands is obtained from http://zsh.dotsrc.org. Note that not all the commands listed on this web page are supported by Symbian.
