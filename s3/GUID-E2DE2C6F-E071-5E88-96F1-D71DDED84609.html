Generic Open Libraries

Generic Open Libraries is a collection of open source libraries that are ported to the Symbian platform. These libraries enable Symbian Developers to easily port POSIX-based applications onto the Symbian platform.

Important: C Standard Library is planned to be deprecated soon. It is recommended that you use the much more compliant and complete Open Environment Core (P.I.P.S.) libraries instead.

  • Open Environment Core (P.I.P.S.)
  • Standard C++ Library (STLport v5)
  • POSIX Realtime Extensions
  • C Standard Library
