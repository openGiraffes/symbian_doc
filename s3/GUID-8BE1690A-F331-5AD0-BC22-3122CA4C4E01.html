Supplementary Service Tutorial

This tutorial describes the tasks to get the supplementary service information with the telephony API for applications.

Note:The supplementary service information is not available if the device is in the flight mode. The call forward and caller identification supplementary services are available only in the GSM and WCDMA networks.
