List of Serial Communications Server Plug-ins

This section provides a list of the CSYs available in Symbian platform. This section does not include Test CSYs or CSYs supplied separately to Symbian platform.

The CSYs available with Symbian platform are modified by Device Creators before they are shipped with a device. These modifications include configuring the CSY to match the hardware ports of the device, and adding security to restrict access to some serial ports.

CSY Filename Description Portname

ecuart.csy

RS232 protocol. On devices this CSY is modified to match the number of ports and any security requirements.

COMM

ircomm.csy

Infra-red protocol for legacy applications. See How to Open a Port with the IrDA Protocol Module: Tutorial.

IrCOMM

btcomm.csy

Bluetooth protocol for legacy applications. See Bluetooth CSY

BTCOMM

ECACM.CSY

USB protocol. See The ACM Server Library Overview.
