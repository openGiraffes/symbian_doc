FreeType Font Rasterizer Component Overview

The FreeType Font Rasterizer is a plug-in to the Font Store. It interprets open font files and generates native Symbian bitmap font characters. It is an optional replaceable component, which means that it may be used or replaced by device creators.

The CBR package name for the FreeType Font Rasterizer component is graphics_freetype.

Architectural relationships

The FreeType Font Rasterizer is an ECOM plug-in to the Font Store.

API summary

No action is required to use the FreeType Rasterizer. It is provided as part of the Symbian platform, installed automatically by the Font Store at startup and has no public API.

It is automatically used to rasterize compatible open fonts.
