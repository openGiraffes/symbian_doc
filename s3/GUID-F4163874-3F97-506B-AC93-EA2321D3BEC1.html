Device Information Tutorial

This tutorial describes how to get the device information using the telephony API for applications.

The telephony API for application is a sub set of the multimode API. The telephony API for applications can be used by any client applications to access the basic telephony functions. The multimode API provides the rich set of telephony functions but it is available only to the handset manufacturers.

    This tutorial describes how to retrieve the phone identification parameters from the device.
  • Subscriber Identity Tutorial
    This tutorial describes how an application gets the subscriber identification information.
  • Flight Mode Status Tutorial
    This tutorial describes how to retrieve the flight mode status of the device.
  • Battery Information Tutorial

  • Battery Charger Tutorial

  • Phone Lock Tutorial
    This tutorial describes how to get the device lock information by the client applications.
