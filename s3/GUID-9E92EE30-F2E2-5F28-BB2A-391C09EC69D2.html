Plug-in Framework

The Plug-in (ECom) framework is a generic framework that provides a common and system-wide mechanism for instantiating a dynamically determined component. The Plug-in (ECom) framework allows the framework or application designers to design interface definitions, plug-in writers to implement interfaces and plug-in users to use the required plug-in.
