Detailed Description

IPsec Encapsulating Security Payload Packet Format.

The function parameters and return values are in host order (except SPI, which is always in network byte order)

RFC-2406

+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+-+ | ----
