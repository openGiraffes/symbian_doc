KRomOnlyResolverUid

const TUidKRomOnlyResolverUid

The UID identifying the ROM-Only CResolver Interface Implementation. This resolver only returns implementations present on R/O internal media (built-in) or upgrades of such implementations on R/W media. It does not return implementations that solely reside on R/W media.
