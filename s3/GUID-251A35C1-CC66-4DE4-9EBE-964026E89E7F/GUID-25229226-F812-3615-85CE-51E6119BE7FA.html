Detailed Description

For building an array on top of TDes8. This is like casting a TDes8 data pointer to an array, but it provides protection against array boundary violations, and some small helpful utilities. The motivation of this class is to help in handling the information accessed by MNetworkInfo interface.

Returns the current length of the array in the number of array elements. The length information is based on the current underlaying descriptor length, which may not always be the desired result.
