Typedef RConnectedAccessories

Type definition RConnectedAccessories consist of connected accessories definitions which are defined in CAccMonitorInfo class. Note: If Pushed onto Cleanup stack, make sure to use ResetAndDestroy Array API just before PopAndDestroy Cleanup stack API, else it causes array elements to leak its memory.
