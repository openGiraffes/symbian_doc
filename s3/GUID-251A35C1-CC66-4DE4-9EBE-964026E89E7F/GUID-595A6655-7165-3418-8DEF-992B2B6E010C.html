Detailed Description

Route information structure.

Used with route options:

  • KSoInetNextRoute

  • KSoInetAddRoute

  • KSoInetDeleteRoute

IPv4 addresses are returned as IPv4-mapped IPv6 addresses qualified with appropriate scope id (Symbian OS 7.0 and later).
