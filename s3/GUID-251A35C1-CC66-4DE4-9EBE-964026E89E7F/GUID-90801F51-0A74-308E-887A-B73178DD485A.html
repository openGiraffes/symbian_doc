    CreateL ( TDes8 &, TMdaClipFormat *, TMdaPackage *, TMdaPackage * )

    IMPORT_C voidCreateL(TDes8 &aDescriptor,
    TMdaClipFormat *aFormat,
    TMdaPackage *aCodec,
    TMdaPackage *aExtra
    )

    Initialises the supplied descriptor to receive the descriptor-based image.

    The function is asynchronous. When the operation is complete, successfully or otherwise, the callback function MMdaImageUtilObserver::MiuoCreateComplete() is called.The descriptor must have been successfully initialised before any conversion operation can start.

    ParameterDescription
    aDescriptorA descriptor to contain the image.
    aFormatThe data format. This is optional.
    aCodecThe codec to use. This is optional.
    aExtraAdditional information which the Media Server may need to recognise the data format. This is optional.

    NewL ( MMdaImageUtilObserver &, CMdaServer * )

    IMPORT_C CMdaImageBitmapToDescUtility *NewL(MMdaImageUtilObserver &aObserver,
    CMdaServer *aServer = NULL
    )[static]

    Constructs a new instance of the bitmap to descriptor-based image conversion utility.

    The function leaves if this object cannot be created.

    ParameterDescription
    aObserverThe image utility observer interface
    aServerA pointer to a server object. The default is NULL which means that one is automatically constructed and owned by this conversion utility object. If not NULL, the client must construct, and be responsible for, the server object. This option may be more efficient if a number of conversion utility objects are to be created.

    Returns: A pointer to the new instance of the bitmap to descriptor-based image conversion utility.
