ls

Lists the directory contents. For more information about using the ls command, search http://www.freebsd.org/cgi/man.cgi by specifying ls as the search keyword and by selecting the FreeBSD 6.2-Release option from the release list.

The following options are not supported by the Symbian version of the ls command:

  • -B

  • -G

  • -H

  • -I

  • -L

  • -P

  • -T

  • -W

  • -Z

  • -b

  • -g

  • -h

  • -i

  • -n

  • -o

  • -w
