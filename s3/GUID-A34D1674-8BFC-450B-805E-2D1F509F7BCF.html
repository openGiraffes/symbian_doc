API Specifications

The API Specifications provide instructions on using a selection of APIs.

The specifications are organized according to the Symbian system model, with each API specification located in the layer and package to which the API belongs.

You can locate the APIs by searching on this page or using the table of contents on the left. You can also use the search to locate an API specification with the name of the API as the search term, or search for any text within the API specifications:

  1. Enter the search term (for example, name of an API) in the Search field.
  2. Click Search scope.
  3. Select Search only the following topics.
  4. Select New.
  5. Select Search only the following topics.
  6. Enter a name for your search scope in the List name field.
  7. Open the library table of contents in the Topics to search and select API specifications as the library section for your search.
  8. Click OK to close New search list window.
  9. Click OK to close Select search scope window.
  10. Click Go to perform your search.
