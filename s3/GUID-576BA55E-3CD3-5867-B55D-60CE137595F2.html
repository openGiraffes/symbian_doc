Introduction to File Stores

File stores are persistent stores that can be used to implement application data files.

There are two types of file store:

  • direct file store, encapsulated by the concrete class CDirectFileStore derived from CFileStore

  • permanent file store, encapsulated by the concrete class CPermanentFileStore derived from CFileStore
