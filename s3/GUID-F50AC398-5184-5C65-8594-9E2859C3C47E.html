File Server Example Code

Describes the example code for the File Server component.

The examples are all text console programs and do not use a GUI environment. The emulator closes when the example terminates.

  • Attributes: Directory Entry Details and Listings
    Describes an example that creates files and directories and retrieves information about them.
  • BasicSession: Hello File Server
    Describes a minimal File Server example.
  • Connecting: Connection to File Server
    Describes a simple example connecting to the File Server and retrieving session information.
  • DriveInfo: Drive Lists, Drive and Volume Information
    Describes an example retrieving drive information.
  • Filenames: Parsing Filenames
    Describes an example extracting information from file paths.
  • Notify: CFileMan Observer
    Describes an example observing a file operation.
  • BinaryFile: Example on file handling
    This is a S60 example demonstrating how to write and read from a file.
  • TextFile Example Overview
    The TextFile example demonstrates how to read from a text file.
  • FileSharing Example Overview
    The FileSharing example is a S60 example that demonstrates how to get a filename and open it and share a file between processes. It consists of two modules FileSharingClient and FileSharingCreator. The FileSharingCreator module uses DoTransferFileL() method, to create and launch the FileSharingClient process. It then transfers filesharing.txt to be read by the client process.
