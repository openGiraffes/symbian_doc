Bearer Mobility

This section gives detailed information on the APIs that enable the email server Message Type Modules (MTM), such as, POP3, IMAP4, and SMTP for non-seamless network bearer mobility. It also gives information on the bearer mobility policy plug-in that is used to receive notifications when a switch to a new bearer occurs, or when the current network connection is lost.
