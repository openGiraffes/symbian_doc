Kernel and Hardware Services Guide

Kernel and Hardware Services performs the fundamental operating system tasks of managing access to device resources.

Note: This documentation now uses the names, such as Kernel and Hardware Services, for components and groups of components that are specified in the system architecture. Some older naming conventions are still used in some documents. Base is a common name that is given to the lower-level components of Symbian platform, which include the Kernel, the File Server, and some device drivers. EKA2 is a common abbreviation for the Kernel Architecture 2 component.
