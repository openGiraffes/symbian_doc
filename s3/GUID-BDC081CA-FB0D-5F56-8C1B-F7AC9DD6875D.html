Writing a Format Plugin

To write a Multimedia Framework (MMF) format plugin, the following needs to be done:

  • Write the appropriate source .cpp and .h files that make up the implementation of the format plugin. See Creating a Format Decode Plugin and Creating a Format Encode Plugin for information on how to do this.

  • Write a resource file defining the formats available in the plugin DLL.

  • Write the appropriate build files.

  • Consider implementation issues.
