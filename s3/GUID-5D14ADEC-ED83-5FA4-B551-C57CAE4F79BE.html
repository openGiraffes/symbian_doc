Zip Compression Library

The Zip Compression Library, EZLib, provides stream and file compression and decompression functionality for the Symbian platforms. EZLib also provides classes to handle the file, stream and buffer requirements of zlib.

  • Zip Compression Library Overview
    The Zip Compression Library, EZLib, provides stream and file compression and decompression functionality for the Symbian platforms.
  • Zip Compression Library Tutorials
    The Zip Compression Library, EZLib compresses files to two formats namely zip and gzip. EZLib also allows clients to decompress data from file archives and memory streams. The compression and decompression tasks are performed iteratively until task completion.
