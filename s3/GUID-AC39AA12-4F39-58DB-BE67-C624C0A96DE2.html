Symbian Platform Fundamentals Example Code

This code demonstrates the fundamental Symbian platform classes and shows basic techniques that can be employed in developing Symbian platform applications.

These examples are simple standalone executables. They use a console to display lines of text to demonstrate the behaviour of the classes.

  • HelloWorld: writes “Hello World!” to the console
  • CClasses: typical ‘C’ class
  • TAndRClasses: typical ‘T’ and ‘R’ type classes
  • MClasses1-3: abstract interface classes
  • CreateStaticDLL and UseStaticDLL: using a statically linked DLL
