00016 <H1>Setup/Teardown</h1> <p>In order to decode audio using
00017 libvorbisfile, a bitstream containing Vorbis audio must be properly
00018 initialized before decoding and cleared when decoding is finished.
00019 The simplest possible case is to use <tt>fopen()</tt> to open a Vorbis
00020 file and then pass the <tt>FILE *</tt> to an <a
00021 href="ov_open.html">ov_open()</a> call.  A successful <a
00022 href="return.html">return code</a> from <a
00023 href="ov_open.html">ov_open()</a> indicates the file is ready for use.
00024 Once the file is no longer needed, <a
00025 href="ov_clear.html">ov_clear()</a> is used to close the file and
00026 deallocate decoding resources.  </b>Do not</b> call <tt>fclose()</tt> on the
00027 file; libvorbisfile does this in the <a
00028 href="ov_clear.html">ov_clear()</a> call.
00029 
00030 <p>
00031 All libvorbisfile initialization and deallocation routines are declared in "vorbis/vorbisfile.h".
