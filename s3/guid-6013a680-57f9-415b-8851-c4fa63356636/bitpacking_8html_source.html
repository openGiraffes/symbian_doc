00016 <h1>Bitpacking Functions</h1>
00017 <p>Libogg contains a basic bitpacking library that is useful for manipulating data within a buffer.
00018 <p>
00019 All the <b>libogg</b> specific functions are declared in "ogg/ogg.h".
