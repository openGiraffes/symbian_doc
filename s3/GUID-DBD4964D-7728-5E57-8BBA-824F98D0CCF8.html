TCP/IP Utilities

This section describes the TCP/IP Utilities collection of components, which provides libraries used by TCP/IPv4/v6 PRT for the DNS and DHCP protocols.

  • Dynamic Host Configuration Protocol
    This section provides the summary of concepts and tutorials related to Dynamic Host Configuration Protocol (DHCP).
  • DND
    DND provides the DNS domain name resolution daemon. It is owned and started by the TCP/IP protocol stack, TCP/IPv4/v6 PRT.

    Summary of concepts and tutorials related to the Domain Name Server (DNS).
